# Real-Time Chat Application

## Overview
This is a comprehensive guide for the Real-Time Chat Application built with React Native, Expo, and React Navigation. The application enables users to communicate in real-time through chat features, providing an engaging user experience on mobile devices.

## Getting Started
To get started with the project, follow the instructions below to set up the environment, install dependencies, and run the application.

### Prerequisites
- Node.js installed on your machine
- Expo CLI installed globally. You can install it using:
  ```bash
  npm install -g expo-cli
  ```

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Catheria29/RealTimechat.git
   cd RealTimechat
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Application
To run the application, use the command:
```bash
expo start
```
This will open a new tab in your default browser where you can scan the QR code with your mobile device to run the app.

### Features
- **Real-time Messaging:** Send and receive messages instantly with WebSocket.
- **User Authentication:** Log in and register using secure methods.
- **User Profiles:** Edit and manage user profiles seamlessly.
- **React Navigation:** Navigate through various screens with ease.
